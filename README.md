# OrderItemPicker
Расширение для 1С:УТ 11.5, добавляющее в форму подбора товаров динамический список остатков выбранной номенклатуры на складах. Остатки отображаются в нижней части формы при активизации строки и получаются из регистра накопления 'Товары на складах'. Реализовано без изменения типовой конфигурации.
