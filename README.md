# Подбор товаров в Заказе. Показывать остатки (УТ 11.5)

---

## Описание задачи
**Расширение** для **1С:Управление торговлей 11.5**, которое **добавляет в форму подбора товаров динамический список остатков выбранной номенклатуры на складах**. Остатки отображаются в нижней части формы при активизации строки и получаются из регистра накопления **"Товары на складах"**. Реализация выполнена **без изменения типовой конфигурации**.

---

## Исходные данные
- **Платформа**: 1С:Предприятие 8.3.27.
- **Редакция УТ**: 11.5.11.70.
- **Тип доработки**: Расширение конфигурации.

---

## Возможности

### 1. Отображение остатков
- При активизации строки в таблице подбора товаров в нижней части формы отображается **динамический список остатков** выбранной номенклатуры.
- Список содержит колонки: **Номенклатура, Характеристика, Склад, Остаток**.

### 2. Динамический отбор
- Остатки фильтруются по выбранной номенклатуре с использованием отбора **"В списке"**.
- Данные актуальны на текущую дату.

---

## Особенности реализации
- **Динамическое создание элементов**: Группа и динамический список создаются программно с использованием **библиотеки [ДФИ](https://github.com/KotovDima1C/DFI)**.
- **Запрос к регистру накопления**: Получение остатков из регистра **"Товары на складах"**.
- **Событие `ПриАктивизацииСтроки`**: Используется для обновления отбора в динамическом списке.

## Как использовать
1. **Установка расширения**:
   - Подключите расширение к конфигурации **1С:Управление торговлей 11.5**.

2. **Работа с подбором**:
   - Откройте документ **"Заказ клиента"**.
   - Нажмите **"Заполнить" → "Подобрать товары"**.
   - Выберите строку с номенклатурой в таблице подбора.
   - В нижней части формы отобразится динамический список с остатками выбранного товара на складах.

---

## Результаты
- Удобное отображение остатков товаров непосредственно в форме подбора.
- Быстрый доступ к информации о наличии товара на складах.
- Упрощение процесса подбора товаров в заказах.
